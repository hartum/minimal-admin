== Installation ==

1. Upload the entire `minimal-admin` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Minimal Admin** menu under **Appearance** in your WordPress admin screen.
